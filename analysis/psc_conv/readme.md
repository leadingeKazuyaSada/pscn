# psc_conv

## 依存するパッケージ

- pyknp==0.4.1

## 必要な外部アプリケーション

- JUMAN++ 2.0.0-rc3

## モジュール

- juman_psc.py
    - JUMAN++ を台本解析用に拡張したサブクラス JumanPsc を定義。
    - 形態素解析の実行時に、なるべくエラーを避ける。
- make_features.py
    - 台本ファイルから各行の特徴量を作るためのモジュール。
- mrph_match.py
    - 形態素列のパターンマッチングをするためのモジュール。
- mrph_test.py
    - テキストファイルが形態素解析可能か調べるためのモジュール。
